Python Snek Game

I originally started this project in 2021 after it was suggested by my father, a data scientist with many years of experience who constantly encourages me to learn and progress.

After many delays caused by this guy, I finished the project in July of 2022.

The game engine I used for this project is the Python Pygame library. It took me some time to get used to it, but eventually I was able to comfortably utilize in some while loops and to create visuals. The .exe file can be found here.

JavaScript Morse Code Translator

Finished on August 23rd, this translator translates English into Morse Code and vice versa. As mentioned in the title, this was created with JavaScript, a language very new to me.

I first started this project by writing the functions in Python, which I later copied in JS by googling everything. After getting stuck and receiving help from the Stack Overflow community, I finished the JS part of the project and started on the HTML.

The most difficult part of the HTML was probably the button and the text areas. The CSS was even harder, trying to position said button and text areas in a way that they would look good if the user resized the browser window, or were on a mobile device instead of a desktop. The translator can be found here.

The Clan Gathering on AoPS

TCG (not the Trading Card Game, though I was into that as well) is a forum on the Art of Problem Solving website, a space where any grade can learn anything.

While I was taking a pre-algebra course on AoPS, I decided to found a forum called "Lionblaze's Den," named after my favorite character (at the time) from the Warriors book series. The forum died some short time after, and I gave up on the idea of creating a forum, until my friends on AoPS surprised me with a new forum they had founded on February 21st, 2020.

We hang around on the forum trying to grow it, until I wasn't able to go online in late May of the same year. When I came back in September, I was shocked to find that a user by the name of eflower2000 had joined during the summer and brought the forum into its golden age. Every thread was so active you could refresh the page every ten seconds and a new message would pop up.

You can join TCG here, and read the rest of my story here.
